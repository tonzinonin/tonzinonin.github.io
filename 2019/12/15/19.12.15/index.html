使用Git上传以及删除远端文件以及nmp下载慢解决方案

2019-12-15 10:40:39

Git

I push文件
以下步骤详细输入
1,git init
2,git add .
(3,git commit -m "first commit"
4,git config --global user.mail "2645331272"
5,git config --global user.name "tonzinonin")
6,git commit -m "first commit"
7,git remote add origin https://github.com/tonzinonin/tonzinonin-code
8,git push -u origin master
    -->if step 8 error Enter：
    [git pull --rebase origin master
    [git push origin master

II delete远程文件
1, git pull origin master 将远程仓库里面的项目拉下来
2, dir  查看有哪些文件夹
3, git rm -r --cached target  删除target文件夹
4, git commit -m '删除了target'  提交,添加操作说明

解决nmp下载慢终极方案：
｛npm config set registry｝ https://registry.npm.taobao.org
        (淘宝镜像)
