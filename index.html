主题下载并启用#

进入命令行，下载 NexT 主题，输入：

Copy
git clone https://github.com/theme-next/hexo-theme-next themes/next

修改站点配置文件 _config.yml，找到如下代码：

Copy
## Themes: https://hexo.io/themes/
theme: landscape

将 landscape 修改为 next 即可。

Hexo基本操作

发表于 2019-12-15

date: 2019-12-15 11:22:37

Hexo Git

1，hexo常见操作符：

hexo new "name"       # 新建文章
hexo new page "name"  # 新建页面
hexo g                # 生成页面
hexo d                # 部署
hexo g -d             # 生成页面并部署
hexo s                # 本地预览
hexo clean            # 清除缓存和已生成的静态文件
hexo help             # 帮助

2，Hexo 设置显示文章摘要，首页不显示全文

Hexo 主页文章列表默认会显示文章全文，浏览时很不方便，可以在文章中插入 <!--more--> 进行分段。

该代码前面的内容会作为摘要显示，而后面的内容会替换为 “Read More” 隐藏起来。

更多详情请见文章：https://zhuanlan.zhihu.com/p/60578464

6,git commit -m "first commit"
7,git remote add origin https://github.com/tonzinonin/tonzinonin-code

    -->if step 8 error Enter：
    [git pull --rebase origin master
    [git push origin master

II delete远程文件
1, git pull origin master 将远程仓库里面的项目拉下来
2, dir  查看有哪些文件夹
3, git rm -r --cached target  删除target文件夹
4, git commit -m '删除了target'  提交,添加操作说明
